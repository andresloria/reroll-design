# Reroll Design — portafolio

Sitio de una página del negocio de diseño y desarrollo web de Andrés Loría (Cartago, Costa Rica). Marca aparte de Reroll Hobby Store.

## Stack
HTML + CSS + JavaScript vanilla. Sin build, sin dependencias. Tipografía Inter + Space Mono vía Google Fonts. Diseño minimalista blanco/negro.

## Archivos
- `index.html` — la página (nav, hero, trabajo, servicios, contacto, footer).
- `css/styles.css` — estilos (cache-bust con `?v=N` al cambiar).
- `js/app.js` — reveal on scroll + año dinámico.
- `assets/` — logo, favicons y app icon (marca Reroll Design, dado d6).

## Local
`python -m http.server 5500` y abrir http://localhost:5500

## Deploy
Vercel (sitio estático, sin configuración). Dominio por definir.
